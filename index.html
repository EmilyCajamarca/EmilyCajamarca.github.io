Concepto
Virtualización

La virtualización es una tecnología que permite crear versiones virtuales de recursos informaticos,como hardware o software , para optimizar de manera eficiente y la flexibilidad de la infraestructura. Permite particionar equipo o un servidor fisico en varias máquinas virtuales, que pueden ejecutar diferentes sistemas operativos o aplicaciones.

Tipos de virtualización

Virtualización de Escritorio

Permite a los usuarios acceder a aplicaciones y sistemas operativos en un solo ordenador a través de máquinas virtuales que se ejecutan en un servidor del centro de datos.Hay dos métodos principales de virtualización de escritorios: local y remoto.La virtualización remota es más robusta y popular, permite acceder a los sistemas operativos y aplicaciones desde un servidor seguro en un centro de datos.La virtualización de escritorios es especialmente útil para empresas de nivel empresarial, ya que permite a los empleados trabajar de forma remota en caso de desastres naturales o problemas de salud.

Virtualización de Aplicaciones

La virtualización de aplicaciones permite a los usuarios acceder a versiones remotas de aplicaciones que no están instaladas en sus propias máquinas. Las aplicaciones virtualizadas funcionan en un entorno separado del sistema operativo, lo que permite un uso sin problemas para el usuario final. Hay dos tipos de virtualización de aplicaciones: remota y en streaming. La virtualización remota de aplicaciones es más popular, ya que permite el acceso a las aplicaciones desde cualquier ubicación autorizada. La virtualización de aplicaciones ofrece ventajas como la reducción de costos y la facilidad de administración.

Virtualización de Servidores

La virtualización de servidores permite dividir un servidor físico en varios servidores virtuales, lo que reduce los costos operativos y mejora el rendimiento de las aplicaciones. Los servidores virtuales comparten recursos como CPU, memoria, almacenamiento y red,extraídos del hipervisor del servidor físico.La virtualización de servidores es ampliamente adoptada en la actualidad, con más del 90% de las empresas utilizando servidores virtuales en lugar de servidores físicos.Esto permite una implementación más rápida y flexible de nuevas máquinas virtuales.

Virtualización de Sistemas de Almacenamiento

La virtualización del almacenamiento permite a las empresas almacenar datos en la nube en lugar de utilizar espacio de almacenamiento físico. Todos los discos de almacenamiento de una empresa se colocan en un pool virtual y se convierten en discos virtuales que se almacenan en un servidor. Esto permite una gestión remota de los recursos de almacenamiento y una mayor flexibilidad en la forma en que se almacenan los datos clave.Hay dos tipos de virtualización de almacenamiento: interna y externa. La virtualización de almacenamiento externo ofrece menos limitaciones en cuanto a la ubicación desde la cual se pueden extraer y almacenar los datos

Virtualización de Redes

La virtualización de redes combina la red física y la virtual para crear un entorno más fácil de administrar.Utilizando software, la virtualización de redes permite desconectar la red virtual del hardware físico y utilizar conmutadores para dirigir el tráfico de la red y gestionar los recursos. Esto es especialmente útil para el desarrollo y prueba de nuevas aplicaciones, ya que permite probarlas en un entorno virtual antes de lanzarlas al público. La virtualización de redes ofrece una mayor flexibilidad y facilidad de administración en comparación con las redes estrictamente físicas

Concepto de docker

Docker es un sistema operativo para contenedores. De manera similar a cómo una máquina virtual virtualiza (elimina la necesidad de administrar directamente) el hardware del servidor, los contenedores virtualizan el sistema operativo de un servidor.

Un contenedor de Docker es una instancia en ejecución de una imagen de Docker. Es un entorno aislado y ligero que contiene todo lo necesario para que una aplicación se ejecute, incluidos el código, las bibliotecas y las dependencias. Los contenedores de Docker son portátiles y se pueden implementar en diferentes sistemas operativosy entornos sin problemas de compatibilidad.
